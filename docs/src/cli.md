# Cli Commands

## Man Pages

Man pages are auto generated, if you've downloaded using nix you have
them ready to go.

```bash
-$ man bm
-$ man bm-go
```

For other linux distributions, the man pages are autogenerated in the build,
they can be located in target/man.

```bash
eza tree
.
├── completions
│   └── bm.bash
└── man
    ├── bm-completions.1
    ├── bm-delete.1
    ├── bm-go.1
    ├── bm-list.1
    ├── bm-save.1
    ├── bm.1
    ├── bookmarks-completions.1
    ├── bookmarks-delete.1
    ├── bookmarks-go.1
    ├── bookmarks-list.1
    ├── bookmarks-save.1
    └── bookmarks.1
```

cli completion is also located in the build files, they are also auto generated
in the build.

## Clap cli

If you're truly an internet connoisseur rust savant, you can
check out the source code directly.

The standard for Clap (argument parser library) is to leave
the structs in cli.rs.

`https://github.com/cowburrs/bookmarks/blob/main/src/cli.rs`

The cli and all commands are defined here. also are all options,
they all contain docstrings explaining their usage.
