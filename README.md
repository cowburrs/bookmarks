# bookmarks - ⚡️ Blazing Fast Bashmarks

---

bookmarks is a bashmarks alternative rewritten in rust (with extra features), it aims to be a simpler replacement to pure bashmarks.

Bashmarks is a personal daily driver, and I wanted to extend it some more.

- [Yazi](https://github.com/sxyazi/yazi) plugin support with fzf!
- Advanced Jaro-Wrinkler string matching
- [Zoxide](https://github.com/ajeetdsouza/zoxide) fallback
- Nix btw.
- Blazingly fast and memory safe.
- Not vibecoded, (check it out).
